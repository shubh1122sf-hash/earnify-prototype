# **App Name**: Earnify Simulator

## Core Features:

- Market: Display a virtual trading dashboard with charts and asset prices.
- Portfolio: Show a simulated portfolio with holdings and balances.
- Login Page: Implement a login page with sign-in with Google.
- Leaderboard: Display a leaderboard of top performing users.
- Stats: Display a statistics page with key metrics.
- Account: A page that shows the user's account information.
- Learn: A page with interactive lessons and tutorials on stocks and crypto.

## Style Guidelines:

- Primary color: #66B2FF (RGB) - A calming blue, reminiscent of established finance apps, promoting trust and competence.  Selected for its versatility, and neither overly saturated nor desaturated.
- Background color: #EBF0F7 (RGB) - A very light, desaturated version of the primary blue to ensure comfortable readability.
- Accent color: #FFA64D (RGB) - A contrasting orange hue for calls to action and interactive elements, providing a sense of energy and motivation.
- Font: 'Inter', a sans-serif font for a clean and modern user interface; suitable for both headings and body text.
- Use clear, professional icons to represent different asset classes and trading functions.
- Employ a clean, intuitive layout with clear separation of concerns and easy navigation.
- Incorporate subtle animations to provide feedback on user interactions and guide users through the application.
